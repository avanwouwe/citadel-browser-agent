MissingApplications Control

What does this control check?

The MissingApplications control verifies that all required applications for your organization are installed on your computer. The specific list of required applications is configured by your IT team based on security requirements, your role, and business needs.

Important: Citadel will tell you which specific application(s) need to be installed. Required applications are typically essential security tools (antivirus, VPN, backup software), mandatory business applications, or compliance tools that protect both you and the organization. Operating without these applications leaves your device vulnerable to security threats and may prevent you from securely accessing company resources.

How to fix this

Installing Required Applications on Windows

‚ö†Ô∏è Contact IT Support:
  • Required applications must be obtained from approved, official sources
  • Your IT team will provide the correct version, license, and installation instructions
  • Some applications require special configuration or company-specific settings
  • Installing the wrong version or from an untrusted source creates security risks
  • Please contact your IT support team to get the required application

Information to provide to IT support:

  • The name of the missing application (from Citadel notification)
  • Your computer name, username, and Windows version
  • Your department and role
  • When you need the application installed by
  • Whether you've had this application before or if this is a new requirement

If IT has provided you with an installer:

  1. Download the installer only from the source provided by IT (email, internal portal, etc.)
  2. Locate the downloaded file in your Downloads folder
  3. Double-click the installer file (usually .exe or .msi)
  4. Click Yes when prompted for administrator permission
  5. Follow all installation wizard instructions
  6. Enter any license keys or configuration details provided by IT
  7. Click Finish when installation is complete
  8. Restart your computer if prompted
Self-Service Options: Your organization may have a self-service portal or software catalog where you can install approved applications yourself. Ask IT support about this option.

Installing Required Applications on macOS

  • Required applications must be obtained from approved, official sources
  • Your IT team will provide the correct version, license, and installation instructions
  • Some applications require special configuration or company-specific settings
  • Installing the wrong version or from an untrusted source creates security risks
  • Please contact your IT support team to get the required application

Information to provide to IT support:

  • The name of the missing application (from Citadel notification)
  • Your computer name, username, and macOS version
  • Your department and role
  • When you need the application installed by
  • Whether you've had this application before or if this is a new requirement

If IT has provided you with an installer:

  1. Download the installer only from the source provided by IT (email, internal portal, etc.)
  2. Locate the downloaded file in your Downloads folder
  3. If it's a .dmg file:
    • Double-click to mount the disk image
    • Drag the application to your Applications folder
    • Eject the disk image when done
  4. If it's a .pkg file:
    • Double-click to run the installer
    • Follow the installation wizard instructions
    • Enter your password when prompted
  5. Launch the application and complete any initial setup
  6. Restart your Mac if prompted
Self-Service Options: Your organization may use an MDM system or self-service portal where you can install approved applications yourself. Ask IT support about this option.

Verifying the fix

After installing the required application, Citadel will automatically verify this control during its next check.

To verify the application is installed:

  1. Open Settings ‚Üí Apps ‚Üí Installed apps
  2. Search for the application name in the search box
  3. The application should appear in the list
  4. Alternatively, check the Start menu - the application should be listed there
  5. Try launching the application to ensure it's working properly

To verify the application is installed:

  1. Open Finder and click Applications
  2. Search for the application name using the search box
  3. The application should appear in the Applications folder
  4. Alternatively, open Launchpad and search for the app
  5. Try launching the application to ensure it's working properly
