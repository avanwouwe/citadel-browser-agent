MaxUptime Control

Control Failed

What does this control check?

The MaxUptime control verifies that your computer has been restarted within the last 7 days. If your system has been running continuously for longer than this period, the control will fail.

How to fix this

Select your operating system below for specific instructions:

Restarting Windows

Method 1: Using the Start Menu

  1. Click the Start button (Windows logo) in the bottom-left corner
  2. Click the Power icon
  3. Select Restart
  4. Wait for your computer to shut down and restart completely

Method 2: Using Keyboard Shortcut

  1. Press Ctrl + Alt + Delete
  2. Click the Power icon in the bottom-right corner
  3. Select Restart

Method 3: From the Desktop

  1. Press Alt + F4 from the desktop (minimize all windows first)
  2. Select Restart from the dropdown menu
  3. Click OK

Restarting macOS

Method 1: Using the Apple Menu

  1. Click the Apple menu () in the top-left corner
  2. Select Restart...
  3. Click Restart in the confirmation dialog
  4. Wait for your Mac to shut down and restart completely

Method 2: Using Keyboard Shortcut

  1. Press Control + Command + Power button
  2. Your Mac will restart immediately

Note: On newer MacBooks with Touch ID, press the Touch ID button instead of the Power button.

Method 3: Scheduled Restart

  1. Open System Settings (or System Preferences)
  2. Go to General > Login Items (or Energy Saver)
  3. Click Schedule... button
  4. Check Restart and set your preferred schedule

Verifying the fix

After restarting your computer:

  1. Wait for Citadel to run its next scheduled check (usually within 1 hour)
  2. Alternatively, you can manually trigger a Citadel check from your system tray/menu bar
  3. The MaxUptime control should now show as Passed
